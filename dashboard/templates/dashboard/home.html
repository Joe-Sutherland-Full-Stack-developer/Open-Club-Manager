Welcome to Open Club Manager

This build is for demonstation purposes.

The site you are about to see is a demo version of the Open source app, populated with example data.

Instructions on how to Install and use Open Club Manager in your own website will be available shortly here:

This demo will give allow you to experience OpenClubManager from the customer's perspective. When prompted, you can use 'test@test.com' to create an account if you do not wish to use a real email address.

*bear in mind that you will not be able to reset your password without a valid email address

View the ReadMe on github for any further information, including admin panel features, which are not available in this demo.

To test the payment system, input card number "4242 4242 4242 4242" with any future date for expiry and any 3 numbers for cvv. The stripe integration is set to test mode so valid cards will not be accepted.
